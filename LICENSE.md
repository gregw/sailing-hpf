## 3. Raw Data Provenance

Race result data reproduced in this database — including elapsed times, boat entries, sail numbers, boat names, and measurement-based handicap values — is sourced from publicly available club race results published by Australian sailing clubs via SailSys and TopYacht software platforms.

This project makes no copyright claim over this raw data. Copyright in the original results may belong to the respective clubs, race organising authorities, or software providers. The data is reproduced here as a matter of public record, in the same spirit as other open sports data projects.

ORC certificate data is sourced from the ORC public data feed (`data.orc.org`) and is used in accordance with ORC's open data policy.

IRC certificate values, where present, are sourced exclusively from publicly published race results (specifically the handicap values published on race result pages by clubs). This project does not use, reproduce, or derive data from the RORC/YCF IRC online TCC listings, which are subject to a restrictive disclaimer prohibiting use for handicap or rating purposes.
